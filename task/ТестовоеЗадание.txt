1. Создать страничку "Task list".
Простой список задач. Нужен для того, чтобы контролировать и не забывать дела, которые нужно сделать за день.

HTML страница состоит из двух блоков:

Первый блок - 
 Блок управления заданиями.
 - Форма ввода + кнопка добавления.
	После ввода текста и нажатия кнопки "ADD TASK" добавляется задание во второй блок(Список задач)
 - Кнопка Remove All - очищает весь список заданий из второго блока. Все элементы удаляются.
 - Кнопка Ready All - все задания в списке отмечаются выполненными. У всех задаинй меняется статус "Не готово" на "Выполнен"

Второй блок - 
 Список всех задач, которые были добавлены через форму(Первый блок).
 Состоит из блоков заданий. Блок задания содержит:
 - Описание задания
 - Кнопка READY - меняет статус задания с "Не готово" на "Выполнено"
 - Кнопка UNREADY - меняет статус задания с "Выполнено" на "Не готово"
 - Кнопка DELETE - удаляет блок с текущим заданием из списка.
 - Статус - содержит текущий статус задания. "Выполено" или "Не готово"

Задание выполнять по приложенному примеру-скриншоту.
Дизайн работы может не совпадать с приложенным, но основные элементы должны соотвествовать заданию.
